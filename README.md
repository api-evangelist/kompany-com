# Kompany.comThis is a repo for managing machine readable artifacts for the [Kompany.com APIs](https://www.kompany.com/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Kompany.com APIs](https://www.kompany.com/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://www.kompany.com/)- [Developer](https://www.kompany.com/)- [Twitter](https://twitter.com/kompanycom)The [APIs.json](https://github.com/api-evangelist/kompany-com/blob/master/apis.json) index provides links to all the relevant links for the Kompany.com APIs, including the OpenAPI and Postman if they are present.
